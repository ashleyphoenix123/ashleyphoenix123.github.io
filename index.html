update

Modify Academic Records


Create, modify, or delete an academic year record. Either the created, modified, or deleted array must be specified.


POST

https://api.github.com/academicyear/update

Parameter
Field Type Description
Optional: created String[]

Array containing 'name', 'start' and 'end' within individual index. (eg created[year_1]['name'] = "year name";)
Optional: modified String[]

Array containing 'uuid', name', 'start' and 'end' within individual index. (eg modified[year_1][uuid] = a61e16eec5b111e59912ba0be0483c18;)
Optional: deleted String[]

Array containing 'uuid' within individual index. (eg deleted[year_1][uuid] = a61e16eec5b111e59912ba0be0483c18;)
PHP : Example usage: 
 Example
Success-Response: JSON

     "status": "ok",

     "status": "error",
     "response": {
         "message": "Missing parameters: name, start, end",
         "code": "invalid-param",
         "http": 400

updateComplete

Modify Academic Records & Periods


Create, modify, or delete an academic year with periods. Either created, modified, or deleted array must be specified. if 'type' equals 'period', it's linked academic year will be changed.


POST

https://api.github.com/academicyear/updateComplete

Using 'created' array parameter
Field Type Description
Optional: created String[]

Array containing 'name', 'start' and 'end' within individual index. (eg. created[year_1]['name'] = "year name";)
Optional: periods String[]

Array within "created" array, containing 'name', 'start' and 'end' within individual index. (eg. created[year_1][periods][period_1][name] = term_name)
Optional: type String

cycle or period. Cycle creates new year with specified periods. Period creates specified periods within an academic year specified by uuid in 'academic_year_uuid'. (eg. created[year_1][type] = "cycle")
Optional: academic_year_uuid Uuid

Only enter if 'type' equals period. (eg. created[year_1][academic_year_uuid] = a61e16eec5b111e59912ba0be0483c18;)
Using 'modified' array parameter
Field Type Description
Optional: modified String[]

Array containing 'name', 'start' and 'end' within individual index. (eg. modified[year_1]['name'] = "year name";)
Optional: periods String[]

Array within "modified" array, containing 'name', 'start' and 'end' within individual index. (eg. modified[year_1][periods][period_1][name] = term_name)
Optional: type String

cycle or period. Cycle edits year. Period edits periods specified by uuid in 'uuid'. (eg. modified[year_1][type] = "cycle")
Optional: uuid Uuid

period uuid if type equals period, academic year uuid otherwise (eg. modified[year_1][uuid] = a61e16eec5b111e59912ba0be0483c18;)
Using 'deleted' array parameter
Field Type Description
Optional: deleted String[]

Array containing next two parameters.
Optional: uuid Uuid

period uuid if type equals period, academic year uuid otherwise (eg. modified[year_1][uuid] = a61e16eec5b111e59912ba0be0483c18;)
Optional: type String

cycle or period. Cycle deletes academic year and all periods. Period deleted a single period.
